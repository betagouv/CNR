Démarche d’accessibilité

Le site du CNR est partiellement conforme.

Il a été conçu pour être accessible au plus grand nombre. Vous devriez donc pouvoir :

  • changer les couleurs, les niveaux de contraste et les polices de caractères ;
  • zoomer jusqu’à 400% sans que le texte ne déborde de l’écran ;
  • naviguer sur toutes les pages du site en utilisant un clavier ;
  • consulter le site web avec un lecteur d’écran.

Nous travaillons actuellement sur le problème suivant :

  • Le module d’acceptation de cookie
