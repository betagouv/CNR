Démarche d’accessibilité

Le site du CNR est en cours d’un audit RGAA 4.1. Il est donc impossible en l’état d’en évaluer la conformité avec le référentiel. Par défaut, le site est donc considéré comme non conforme.

Il a cependant été conçu pour être accessible au plus grand nombre. Vous devriez donc pouvoir :

  • changer les couleurs, les niveaux de contraste et les polices de caractères ;
  • zoomer jusqu’à 400% sans que le texte ne déborde de l’écran ;
  • naviguer sur toutes les pages du site en utilisant un clavier ;
  • consulter le site web avec un lecteur d’écran.

Nous travaillons actuellement sur les problèmes suivants :

  • changer les accordéons de sélection sur la page du formulaire d’inscription pour que leur comportement soit plus naturel au clavier ;
  • ajouter des libellés aux questions des formulaires thématiques.
